## 一、商业路径建议

### 推荐路径：PLG（Product-Led Growth）+ 增值订阅

```
免费版（社区版）  →  Pro 个人版  →  Team 团队版  →  Enterprise 企业版（远期）
      ↓                ↓                ↓
   个人用户         独立开发者         小微团队
   跑通流程        提效为主           协作为主
```

**理由**：
1. **价格敏感 ≠ 不付费** —— 独立开发者愿为「省时间」付费（参考 Cursor、Raycast、Linear）
2. **PLG 边际成本低** —— 获客靠产品本身，不是销售
3. **飞轮清晰** —— 免费用户用顺手 → 推荐同事 → 团队升级

### 备选路径对比

| 路径 | 优势 | 风险 | 适合 |
|---|---|---|---|
| **PLG + 订阅**（推荐）| 启动快、飞轮清晰 | LLM 成本失控 | 想 6 个月内出付费用户 |
| **开源核心 + 商业模块** | 社区贡献、生态 | 竞品复制快 | 想做平台、做生态 |
| **纯私有化 license** | 高客单价、企业友好 | 销售周期长 | 有人脉/渠道资源 |
| **完全免费 + 增值服务** | 用户最多 | 商业化最慢 | 不急着赚钱 |

**我的建议**：先 PLG 跑 6 个月，如果增长健康，Q4 再决定要不要开源核心。

| P0 | 内置项目模板（Web / API / 小程序 / 数据脚本） | 🔜 |

## 三、关键决策点（建议你本月内拍板）

### 决策 1：MVP 核心场景

选项：
- **A. 独立开发者自用**（单人）—— 起步快，需求清晰
- **B. 小团队协作**（多人）—— 客单价高，但需求复杂
- **C. 模板分发**（开发者市场）—— 长期价值高，但冷启动难

**推荐 A → 后续往 B**。原因：技术债务低，能快速验证付费。

### 决策 2：LLM 默认策略

选项：
- **A. 全本地 Ollama**（隐私好、成本低、模型弱）
- **B. 默认云端**（体验好、成本高）
- **C. 混合（用户自选 + 默认推荐）**

**推荐 C**：默认推荐 Ollama 本地 + 一键切换云端。理由：给用户选择权，我们不背成本风险。

### 决策 3：开源策略

选项：
- **A. 完全闭源**（保护壁垒）
- **B. 核心开源 + 商业模块**（社区驱动 + 商业化）
- **C. 完全开源**（靠服务/支持赚钱）

**推荐 B**：核心（多 Agent 协作框架）MIT 开源，云端服务 / 企业版收费。理由：开发者天然倾向开源产品，能加速获客；同时云端版本有数据、协作、模型调用等独特价值。

### 决策 4：融资节奏

选项：
- **A. 自筹到 $50K MRR 再考虑**
- **B. 早期天使 $50-200K 加速 6 个月**
- **C. 立即启动 A 轮（不推荐）**

**推荐 A → B 触发**：做到 $5K MRR 触发天使。理由：先证明 PMF，再拿钱加速。

## 四、竞争壁垒思考

| 壁垒类型 | 可行性 | 说明 |
|---|---|---|
| **流程标准** | ⭐⭐⭐ | 需求→设计→评审→开发→评审→测试→评审 是工业实践，难复制 |
| **数据飞轮** | ⭐⭐⭐ | 用户的项目数据沉淀，迁移成本高 |
| **网络效应** | ⭐⭐ | 团队 / 模板社区效应，缓慢但稳 |
| **品牌** | ⭐⭐ | AI 原生项目管理的「专家品牌」 |
| **技术专利** | ⭐ | 不靠专利 |

**核心壁垒**：把"标准研发流程"做成行业标准 + 用户数据沉淀。

## 五、风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| LLM 成本失控 | 高 | 高 | 多模型路由 + 缓存 + 限额 |
| 大厂下场（飞书/钉钉/Notion AI） | 中 | 高 | 垂直做深 + 流程标准化 |
| AI 输出质量不稳定 | 中 | 中 | 人工评审环节 + 反馈学习 |
| 独立开发者付费率低 | 中 | 中 | 长期靠团队 / 企业 |
| 隐私合规 | 中 | 中 | 私有化部署版本 |
| 技术债务 | 中 | 中 | Phase 2 强制云化重构 |

## 七、给创始人的提醒

1. **先验证 PMF，再谈规模** —— 别被 10000 注册用户迷惑，关键看 5% 付费率
2. **LLM 成本是定时炸弹** —— 一开始就设计配额和缓存
3. **独立开发者的口碑传播极快** —— 用心服务前 100 个用户
4. **不要过早平台化** —— 先把核心流程做透
5. **保留产品质感** —— 工具类产品的设计感是壁垒之一

## 附录 B：成本估算（独立开发者版 SaaS）

**单用户月成本估算**：
- LLM（混合模式）：$1-3
- 基础设施（摊销）：$0.5
- 客服（摊销）：$0.5
- 合计：**$2-4 / 月 / 用户**

- Pro $9/月用户：毛利率 55-77%
- Team $29/月/5 人：单用户 $5.8，毛利率 31%（但获客成本低）

**目标毛利率**：60%+ 才健康
